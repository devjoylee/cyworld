## 프로젝트 구현 사항

graphql에 특화된 라이브러리인 apollo로 API 데이터 사용

- C : `createBoard`
- R : `fetchBoard`(single) / `fetchBoards`(list)
- U : `updateBoard`
- D : `deleteBoard`
